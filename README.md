## 1. 시작하기 — 회원가입과 로그인

### 처음 사용할 때

1. 브라우저에서 SOERP 주소로 접속합니다. 로그인하지 않은 상태면 자동으로 로그인 화면(`login.html`)이 열립니다.
2. **이메일**과 **비밀번호**를 입력하고 **회원가입**을 누릅니다.
3. 이메일 인증이 켜져 있다면 메일함에서 인증 링크를 클릭한 뒤 돌아옵니다.
4. 같은 이메일/비밀번호로 **로그인**하면 대시보드로 이동합니다.

### 알아두면 좋은 점

- **계정별로 데이터가 완전히 분리됩니다.** 다른 계정으로 로그인하면 내 업무는 보이지 않습니다.
- **어느 기기에서든 같은 데이터를 봅니다.** 사무실 PC에서 등록한 업무를 집이나 휴대폰에서 그대로 확인·수정할 수 있습니다.
- 로그아웃은 왼쪽 사이드바 **맨 아래 로그아웃 버튼**을 누르면 됩니다. 그 위에 현재 로그인한 이메일이 표시됩니다.

## 2. 화면 구성 익히기

### 왼쪽 사이드바 — 메뉴

| 메뉴 | 하는 일 |
|---|---|
| **Dashboard** | 오늘 할 일과 전체 진행 현황 요약 |
| **Tasks** | 전체 업무 목록 — 검색·필터·정렬·등록·삭제 |
| **Calendar** | 월별 달력으로 마감일 확인 |
| **Templates** | 업무 유형별 체크리스트 템플릿 관리 |
| **Settings** | 백업/복원, 데이터 초기화, 앱 정보 |

> 휴대폰이나 좁은 화면에서는 사이드바가 숨겨집니다. 왼쪽 위 **☰ 메뉴 버튼**을 눌러 여세요.

### 위쪽 헤더

- **검색창** — 업무명을 입력하고 **Enter**를 누르면 어느 화면에 있든 업무 현황 페이지로 이동하며 검색 결과를 보여줍니다.
- **오늘 날짜와 시각** — 매분 자동으로 갱신됩니다.
- **＋ 새 업무 추가** — 어느 화면에서든 바로 업무를 등록할 수 있습니다.

## 3. 대시보드 — 오늘 할 일 확인

로그인하면 가장 먼저 보이는 화면입니다. 아침에 한 번 훑어보는 용도로 만들어졌습니다.

### 맨 위 요약 카드 4개

| 카드 | 세는 기준 |
|---|---|
| **오늘 업무** | 마감일 또는 업무 날짜가 오늘인 미완료 업무 |
| **진행 중** | 상태가 '진행 중'인 업무 |
| **확인 대기** | 상태가 '확인 대기'인 업무 (회신을 기다리는 것들) |
| **이번 주 마감** | 7일 이내 마감 예정인 미완료 업무 |

### 왼쪽 — 업무 목록

- **오늘의 업무** — 오늘 처리해야 할 것들
- **마감 지난 업무** — 🔴 마감일이 지났는데 아직 완료·취소되지 않은 업무. **없으면 이 영역 자체가 표시되지 않습니다.**
- **마감 임박 업무** — 🟡 3일 이내 마감 예정
- **확인 대기 업무** — 상대방 회신을 기다리는 업무

각 업무 카드를 클릭하면 상세 화면으로 이동합니다.

### 오른쪽 — 현황 통계

- **전체 진행률** — 업무 완료율과 체크리스트 완료율
- **상태별 / 카테고리별 / 우선순위별 현황**
- **최근 완료 업무** — 마지막으로 완료한 5건

## 4. 업무 등록하기

헤더나 업무 현황 화면의 **＋ 새 업무 추가**를 누르면 등록 창이 열립니다.

### 입력 항목

| 항목 | 필수 | 설명 |
|---|:---:|---|
| **업무명** | ✅ | 예: `○○아파트 계약보증서 발급` |
| 카테고리 | | 입찰 / 계약 / 회계·청구 / 보증서 / 증명서 / 문서정리 / 연락·확인 / 기타 |
| 상태 | | 할 일 / 진행 중 / 확인 대기 / 완료 / 취소 |
| 우선순위 | | 긴급 / 높음 / 보통 / 낮음 |
| 요청자/지시자 | | 예: `김부장님` |
| 현장명 | | 예: `○○아파트` |
| 거래처명 | | 예: `○○관리사무소` |
| 금액 | | 숫자만 입력하면 자동으로 천 단위 콤마가 붙습니다 |
| 업무 날짜 | | 업무를 진행하는 날 |
| 마감일 | | **대시보드·달력·알림의 기준이 되는 가장 중요한 날짜** |
| 확인사항 | | 예: `담당자 회신 대기` — 카드와 상세에 눈에 띄게 표시됩니다 |

### 체크리스트 — 템플릿 활용이 핵심

카테고리를 고르면 **그 카테고리의 템플릿 불러오기 버튼**이 나타납니다.
예를 들어 카테고리를 `계약`으로 두면 `낙찰 업무 템플릿 불러오기`, `착공 업무 템플릿 불러오기`, `준공 업무 템플릿 불러오기` 버튼이 뜹니다.

- 버튼을 누르면 그 템플릿의 항목이 체크리스트로 한 번에 들어옵니다.
- **이미 체크리스트에 항목이 있으면 "덮어쓰기" 확인 창이 뜹니다.** 기존 항목이 사라지니 주의하세요.
- 불러온 뒤 필요 없는 항목은 지우고, 필요한 항목은 추가하면 됩니다.
- 항목 입력칸에 직접 쓰고 **Enter** 또는 **추가** 버튼으로 하나씩 넣을 수도 있습니다.

마지막으로 **업무 등록**을 누르면 저장됩니다.

> 💡 작성 중에 창을 닫으려 하면 "작성 중인 내용이 사라집니다" 확인 창이 뜹니다. **Esc** 키로도 닫기를 시도할 수 있습니다.

## 5. 업무 현황 — 찾고 거르고 정렬하기

**Tasks** 메뉴에서 등록한 모든 업무를 봅니다. 제목 아래에 `전체 ○건 · 표시 ○건`으로 필터 결과가 표시됩니다.

### 검색

검색창에 입력하면 **입력하는 즉시** 결과가 좁혀집니다. 검색 대상은 다음과 같습니다.

- 업무명, 요청자, 현장명, 거래처명, 확인사항, **그리고 메모 내용까지**

> 💡 메모까지 검색되므로 "지난번에 통화한 그 건"처럼 메모에만 적어둔 내용으로도 찾을 수 있습니다.

### 필터

- **완료 여부** — 전체 보기 / 미완료만 / 완료만
- **상태** / **카테고리** / **우선순위** — 각각 원하는 값으로 좁히기

### 정렬

- **마감일순** (기본) — 마감일이 없는 업무는 맨 뒤로 갑니다
- **생성일순** — 최근에 등록한 것부터
- **우선순위순** — 긴급 → 높음 → 보통 → 낮음

### 보기 방식 전환

오른쪽 끝 두 버튼으로 전환합니다.

- **카드형** — 체크리스트 진행률 막대가 크게 보여 진행 상황 파악에 좋습니다.
- **테이블형** — 업무명·카테고리·상태·우선순위·현장/거래처·마감일·금액·진행률을 한 줄로 봅니다. 건수가 많을 때 편합니다.

### 목록에서 바로 처리하기

- **상태 변경** — 카드 아래(또는 표의 상태 칸)의 드롭다운에서 바로 바꿉니다. 상세 화면에 들어갈 필요가 없습니다.
- **삭제** — 카드 오른쪽 위(또는 표 맨 오른쪽)의 🗑 버튼. 확인 창이 한 번 뜹니다.
- **상세 보기** — 카드나 행의 아무 곳이나 클릭

### 색으로 알아채기

- 🔴 **빨간 마감일 + "마감 지남"** — 기한이 지난 업무
- 🟡 **"오늘 마감"** — 오늘까지인 업무
- **테두리가 강조된 카드** — 우선순위가 '긴급'인 미완료 업무

## 6. 업무 상세 — 체크리스트와 메모

업무를 클릭하면 열리는, 실제 작업이 이루어지는 화면입니다.

### 위쪽 버튼

- **뒤로가기** — 이전 화면으로
- **완료 처리** — 상태를 '완료'로 바꾸고 완료일을 자동 기록합니다 (이미 완료된 업무에는 표시되지 않습니다)
- **수정** — 등록 때와 같은 창에서 내용을 고칩니다
- **삭제** — 확인 후 삭제되며 업무 목록으로 돌아갑니다

### 체크리스트 (가장 많이 쓰는 기능)

- **동그라미를 클릭**하면 체크/해제됩니다. **진행률(%)이 즉시 자동 계산**됩니다.
- **✏️ 연필** — 항목 글자 수정. 수정 중 **Enter**로 저장, **Esc**로 취소
- **🗑 휴지통** — 항목 삭제
- 아래 입력칸에 쓰고 **Enter** 또는 **추가**로 새 항목 등록

### 메모 (댓글 방식)

업무 진행 상황을 시간순으로 기록하는 공간입니다.

- 아래 입력칸에 쓰고 **등록** 버튼 또는 **Ctrl + Enter**(맥은 ⌘ + Enter)로 저장
- 각 메모에 **작성 일시가 자동으로 남습니다** — `6월 12일 (금) 14:32`
- 오래된 메모가 위, 최신 메모가 아래에 쌓입니다
- 🗑 버튼으로 개별 삭제

> 💡 통화 내용, 회신 여부, 특이사항을 그때그때 남겨두면 나중에 검색으로 찾을 수 있습니다.

### 오른쪽 정보란

- **업무 정보** — 요청자, 현장명, 거래처명, 금액, 업무 날짜, 마감일, 완료일
- **이력** — 생성일, 마지막 수정일

## 7. 달력으로 일정 보기

**Calendar** 메뉴에서 월별 일정을 확인합니다.

- 각 업무는 **마감일** 위치에 표시됩니다. 마감일이 없으면 **업무 날짜** 기준입니다.
- 하루에 여러 건이면 **우선순위 높은 순으로 최대 4건**만 보이고, 나머지는 `+○건`으로 표시됩니다.
- 업무 이름을 클릭하면 바로 상세 화면으로 이동합니다.
- 색은 **상태**를 뜻하며, 완료·취소된 업무는 흐리게 보입니다.
- **← / →** 로 이전·다음 달 이동, **오늘** 버튼으로 이번 달로 복귀
- 오늘 날짜는 강조 표시되고, 일요일은 빨강 / 토요일은 파랑으로 구분됩니다.

## 8. 업무 템플릿 관리

**Templates** 메뉴에서 반복 업무의 체크리스트를 미리 만들어 둡니다. 업무 등록 시 한 번에 불러오는 그 목록입니다.

### 기본으로 제공되는 템플릿 9종

| 템플릿 | 카테고리 | 내용 |
|---|---|---|
| 입찰 업무 템플릿 | 입찰 | 공고문 확인 → 제출 → 결과 확인 (9개 항목) |
| 낙찰 업무 템플릿 | 계약 | 낙찰 확인 → 계산서 수령 → 계약서 초안 (3개) |
| 착공 업무 템플릿 | 계약 | 착공일자·계약서 원본·현장대리인 확인 (5개) |
| 준공 업무 템플릿 | 계약 | 준공일자·잔여 기성금·하자보증서 (4개) |
| 보증서 업무 템플릿 | 보증서 | 전문건설공제조합 신청 → 결제 → 보관 (11개) |
| 회계/청구 업무 템플릿 | 회계/청구 | 세금계산서 발행 → 입금 확인 (8개) |
| 증명서 업무 템플릿 | 증명서 | 발급 → 제출 → 보관 (8개) |
| 문서정리 업무 템플릿 | 문서정리 | 스캔 → PDF 변환 → 메일 송부 (8개) |
| 연락/확인 템플릿 | 연락/확인 | 연락 → 회신 확인 → 후속 처리 (7개) |

### 할 수 있는 일

- **항목 추가** — 카드 아래 입력칸에 쓰고 Enter 또는 추가
- **항목 수정/삭제** — 각 항목의 ✏️ / 🗑 버튼
- **이름·설명 수정** — 카드 위쪽 ✏️ 버튼
- **템플릿 삭제** — 카드 위쪽 🗑 버튼
- **복사** — 템플릿 항목을 텍스트로 클립보드에 복사
- **새 템플릿 추가** — 목록 맨 끝의 카드에서 카테고리·이름·설명을 입력해 만듭니다. 만든 뒤 항목을 채우세요.
- **기본값 복원** — 기본 템플릿 카드 아래의 링크로 그 템플릿만 원래대로 되돌립니다
- **기본 템플릿 복원** — 오른쪽 위 버튼. 삭제했던 기본 템플릿 9종을 다시 불러옵니다 (내가 만든 템플릿은 지워지지 않습니다)

> ⚠️ **템플릿은 업무 데이터와 달리 이 브라우저에만 저장됩니다.** 다른 PC에서 로그인하면 그곳의 템플릿은 기본 상태입니다. 브라우저 데이터를 지우면 템플릿 수정 내용도 사라지니, 공들여 만든 템플릿은 따로 메모해두는 것을 권합니다.

- **데이터 초기화** — 🔴 **모든 업무가 삭제됩니다. 되돌릴 수 없습니다.**

### 앱 정보

현재 로그인한 이메일, 저장된 업무 건수, 마지막 업데이트 시각을 확인할 수 있습니다.

> 💡 **권장 습관**: 월말이나 분기말에 한 번씩 **JSON 내보내기**로 백업 파일을 받아두세요.

## 10. 용어 정리

### 상태 5가지

| 상태 | 언제 쓰나 |
|---|---|
| **할 일** | 아직 시작하지 않음 (등록 시 기본값) |
| **진행 중** | 지금 처리하고 있음 |
| **확인 대기** | 내 할 일은 끝났고 상대방 회신·승인을 기다리는 중 |
| **완료** | 끝남 — 완료일이 자동 기록됩니다 |
| **취소** | 진행하지 않기로 함 — 마감 지남 계산에서 제외됩니다 |

### 우선순위 4가지

**긴급** > **높음** > **보통**(기본값) > **낮음**
'긴급'인 미완료 업무는 목록에서 눈에 띄게 강조됩니다.

### 카테고리 8가지

입찰 · 계약 · 회계/청구 · 보증서 · 증명서 · 문서정리 · 연락/확인 · 기타

### 날짜 두 가지 구분

- **업무 날짜** — 업무를 실제로 진행하는 날
- **마감일** — 반드시 끝내야 하는 날. **대시보드 알림과 달력이 이 날짜를 기준으로 동작하므로, 되도록 채워 넣으세요.**

## 12. 자주 묻는 질문

**Q. 다른 PC나 휴대폰에서도 볼 수 있나요?**
네. 같은 이메일로 로그인하면 업무 데이터는 그대로 보입니다. 단, **업무 템플릿은 기기별로 따로 저장**되므로 새 기기에서는 기본 템플릿 상태입니다.

**Q. 실수로 업무를 삭제했어요.**
복구 기능은 없습니다. 백업 파일이 있다면 **설정 → JSON 가져오기**로 복원할 수 있습니다(단, 그 이후 추가한 업무는 함께 사라집니다).

**Q. 마감일을 안 넣으면 어떻게 되나요?**
등록은 되지만 마감 임박·마감 지남 알림에 잡히지 않고, 달력에서는 '업무 날짜' 자리에 표시됩니다. 마감일 정렬 시에는 목록 맨 뒤로 갑니다.

**Q. 완료한 업무가 목록에서 사라졌어요.**
필터의 **완료 여부**가 '미완료만'으로 되어 있는지 확인하세요. '전체 보기'나 '완료만'으로 바꾸면 보입니다.

**Q. 상태를 '완료'에서 다른 것으로 되돌리면요?**
완료일 기록이 지워지고 다시 미완료 업무로 집계됩니다.

**Q. 같은 카테고리에 템플릿을 여러 개 만들어도 되나요?**
됩니다. 기본 제공되는 '계약' 카테고리도 낙찰·착공·준공 3종으로 나뉘어 있습니다. 업무 등록 시 해당 카테고리의 템플릿 버튼이 모두 표시됩니다.

**Q. 다른 사람이 내 업무를 볼 수 있나요?**
없습니다. 계정별로 데이터가 분리되어 있어 본인 계정의 업무만 조회됩니다.

**Q. 저장 버튼이 따로 없나요?**
체크리스트 체크, 상태 변경, 메모 등록은 **누르는 즉시 저장**됩니다. 업무 등록/수정 창에서만 마지막에 '업무 등록' 또는 '수정 저장' 버튼을 눌러야 합니다.

**Q. "저장에 실패했습니다. 네트워크를 확인해주세요"가 떠요.**
인터넷 연결이 끊겼을 때 나타납니다. 연결을 확인한 뒤 다시 시도하세요. 데이터는 온라인 데이터베이스에 저장되므로 오프라인에서는 저장되지 않습니다.
